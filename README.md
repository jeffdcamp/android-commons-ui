Android Commons UI Library
==========================

Share Android UI Library

**Features**

* Top AppBar
  * Bar icons / Overflow menu
  * Navigation Drawer
  * Navigation Rail
* Dialogs
  * Input
  * Radio
  * Message
  * MenuOptions
  * MultiSelect
  * DatePicker
  * TimePicker
* Settings
* AutoSizeText
* DayNightTextFields
* DropDownMenu
* Navigation
  * Route definitions
  * Compose Routes
  * Logging
